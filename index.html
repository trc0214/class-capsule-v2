使用指南

🔧 初次設置

  1. 在最上方輸入您的 Azure Speech API 密鑰（japaneast 區域）
  2. 輸入您的 Gemini API 密鑰
  3. 點擊「保存設定」，密鑰會被安全地存儲在瀏覽器本地

🎙️ 錄音使用

  1. 確認麥克風權限已授予
  2. 點擊「開始錄音」開始即時轉錄
  3. 邊講邊轉錄，系統會連續識別您的語音
  4. 完成後點擊「停止錄音」

✨ 生成摘要

  1. 選擇您偏好的摘要風格和長度
  2. 點擊「生成 Gemini AI 摘要」
  3. AI 會自動生成結構化筆記

💡 重要提示

  • API 密鑰存儲在您的瀏覽器本地，不會上傳到任何服務器
  • 確保網絡連接穩定以支持 2 小時的連續錄音
  • 支持中文和資工專有名詞的識別
  • 如遇識別中斷，系統會自動重新連接
